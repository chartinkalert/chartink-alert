Setup in 60 seconds

  1. Open @notifyu_bot on Telegram and send /start.
  2. Send /myuid and copy the webhook URL it gives you.
  3. Paste that URL into your Chartink alert webhook field.
  4. Trigger the alert → you’ll receive a message instantly.
Example webhook format:
https://notifyu.me/chartink?uid=YOUR_UID&key=YOUR_KEY
Useful commands:
/myuid - show webhook URL
/newuid - rotate UID + key
/unlink - remove link
